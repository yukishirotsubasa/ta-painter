# sidebar3 — 側邊欄已畫線清單（檢視/刪除）

## 說明

側邊欄「畫線區塊」（可折疊）。列出目前所有畫線，點清單項可選取（高亮圖上對應線段），每項附刪除鈕可刪除單條。複用 drawing6 曝光的 `getLines()`/`onLinesChange`/`deleteLine(id)`。此區塊**僅提供檢視與刪除**，不含畫線開關與改色。取代原本畫布點擊選取刪除的路徑（已於 drawing6 移除），觸控與桌面通用。折疊畫線清單、或折疊整個側邊欄時，**自動取消目前選取（清除線段高亮）**。

## 依賴

drawing6

## 驗收方式

1. 畫線後側邊欄畫線區塊即時列出對應線段。
2. 點清單項高亮圖上對應線段；點刪除鈕移除該線且圖上消失。
3. 觸控裝置亦可透過清單刪除單條線（不需鍵盤）。
4. 折疊畫線清單、或折疊整個側邊欄時，自動取消選取並清除圖上線段高亮。
