# responsive3 — 觸控畫線手勢最終調整

## 說明

在真機環境下針對觸控裝置微調畫線模式與圖表原生 pan/zoom 手勢的互斥切換，確保單指拖曳（平移圖表）與畫線模式下的點擊放置端點不互相干擾，畫線模式有清楚的視覺提示。

## 依賴

responsive1, drawing2

## 驗收方式

1. 真機（iOS Safari + Android Chrome）測試：非畫線模式下單指拖曳可正常平移圖表、雙指可正常縮放。
2. 真機測試：進入畫線模式後，單指點擊能正確放置端點，且不會誤觸發圖表平移。
3. 畫線模式開啟/關閉時 UI 提示清楚可辨識，使用者不會誤以為手勢失靈。
