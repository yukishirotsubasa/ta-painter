# data7 — 資料源切換與官方源慢速提示

## 說明

App 預設資料源由現行寫死 TWSE（`App.tsx:55`）改為 **Yahoo**（單次請求快查）。提供資料源切換：Yahoo / 官方，官方依市場別路由 `TwseProvider`（上市）/ `TpexProvider`（上櫃）。在資料層加入 `estimateSlow(range)`（例如查詢月數 > 3 視為慢）供側邊欄（sidebar2）呈現逐月抓取等待提示；request 次數限制屬**程式內部節流/限流**（避免頻繁查詢被上游封鎖），**不對使用者顯示提示**。依使用者決策，量能依來源原樣顯示，**不做跨源標示或正規化**（移除原「自動選源」與「標示來源」構想）。

## 依賴

無。

## 驗收方式

1. 預設以 Yahoo 查詢，長區間一次取回。
2. 切官方源查長區間時顯示逐月抓取等待提示；短區間不顯示不必要提示。request 次數限制為程式內部節流，不對使用者顯示。
3. 上市/上櫃代號在官方源分別走 TWSE/TPEx 正確取得資料。
